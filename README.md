# Задание

На входе папка/-ки с картинками.

Требуется написать код, который по запросу из списка папок, собирает картинки из папок в один tiff файл.

### Пример.

На входе список: ['1388_12_Наклейки 3-D_3']

На выходе файл: Result.tiff
